Local
-----

Local services can be further categorised as either mandatory or optional.

:doc:`Mandatory <mandatory/mandatory>` services are compulsory services run at each node in the SRCNet network. These
services ensure fundamental operations and baseline functionality.

:doc:`Optional <optional/optional>` services are non-mandatory services run at each node in the SRCNet network. These
services add to the mandatory functionality required at each node.
